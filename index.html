Linux下OGG搭建

date: 2016-05-06 17:15:07

OGG下载地址: http://www.oracle.com/technetwork/cn/testcontent/index-100295-zhs.html

###源端：
操作系统: 64位redhat 6.2

OGG版本: 11.2.1.0.1

###目标端：
操作系统: 64位redhat 6.2

OGG版本: 11.2.1.0.1

##数据库配置
开启归档

打开数据库级别的补充日志(supplemental log)

##源端配置
Manager进程

Extract进程

extract ext1
dynamicresolution
userid ogg,password ogg
setenv(ORACLE_SID=orcl)
exttrail /ogg/dirdat/et

table scott.dept;

GGSCI> add extract ext1,tranlog,begin now
GGSCI> add exttrail /ogg/dirdat/et, extract ext1, MEGABYTES 20

Pump进程

extract pump1
rmthost 192.168.54.134,mgrport 7500
userid ogg,password ogg
setenv(ORACLE_SID=orcl)
rmttrail /ogg/dirdat/et

table scott.dept;

GGSCI> add extract pump1, exttrailsource /ogg/dirdat/et
GGSCI> add rmttrail /ogg/dirdat/et, extract pump1, megabytes 20

##目标端配置
Manager进程

Replicat进程

replicat repl1
userid ogg,password ogg
assumetargetdefs
reperror default,discard
discardfile /ogg/dirrpt/replicat1.dsc, append, megabytes 50
dynamicresolution

map scott.dept, target scott.dept;

GGSCI> add replicat repl1, exttrail /ogg/dirdat/et, checkpointtable ogg.checkpoint
