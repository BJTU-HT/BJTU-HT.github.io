外键导致OGG进程异常

发表于 2016-05-24   |  

tags: WARNING OGG-00869, ORA-02291

date: 2016-05-24 16:46:30

OGG目的端复制进程异常，查看进程报告，报如下错误：

2016-05-24 15:44:27 WARNING OGG-00869 OCI Error ORA-02291: integrity constraint (NM_BAK.SYS_C0053657) violated - parent key not found (status = 229
1). INSERT INTO “NM_BAK”.”SNSLOTTYPE” (“ID”,”SNSUBNETYPEID”,”CASCADESEQUENCE”,”SEQUENCE”,”DISPLAYLABEL”,”LEFTTOPX”,”LEFTTOPY”,”WIDTH”,”COMMENTS”,”VER
SION”,”HEIGHT”,”MIGDATE”,”SLOTTYPEID”) VALUES (:a0,:a1,:a2,:a3,:a4,:a5,:a6,:a7,:a8,:a9,:a10,:a11,:a12).

2016-05-24 15:44:27 WARNING OGG-01004 Aborted grouped transaction on ‘NM_BAK.SNSLOTTYPE’, Database error 2291 (OCI Error ORA-02291: integrity const
raint (NM_BAK.SYS_C0053657) violated - parent key not found (status = 2291). INSERT INTO “NM_BAK”.”SNSLOTTYPE” (“ID”,”SNSUBNETYPEID”,”CASCADESEQUENCE
“,”SEQUENCE”,”DISPLAYLABEL”,”LEFTTOPX”,”LEFTTOPY”,”WIDTH”,”COMMENTS”,”VERSION”,”HEIGHT”,”MIGDATE”,”SLOTTYPEID”) VALUES (:a0,:a1,:a2,:a3,:a4,:a5,:a6,:
a7,:a8,:a9,:a10,:a11,:a12)).

发现是目标端的表NM_BAK.SNSLOTTYPE上的外键约束NM_BAK.SYS_C0053657引起的，将这个外键约束删除就好了。
